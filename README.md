# 多模态AI认知康复训练系统

利用**个性化生成模型 + 多模态AI技术**，为认知障碍患者（尤其是阿尔茨海默病患者）提供高度定制化的视觉、视频、音视频及VR康复内容，大幅提升患者参与积极性和康复效果。

## Part 1：多维度认知康复训练刺激内容生成

### 1. 情景回忆内容生成
基于患者少量旧照片 + ID一致的个性化模型，生成患者在熟悉场景中的高保真图像（购物、做饭、看书、散步等），精准激发自传式记忆。

<img src="./image/sence.png" width="600"/>

### 2. 年龄定制化照片生成
一键生成患者少年、青年、中年、老年四个阶段的照片，帮助系统性回忆人生轨迹与亲友关系。

<img src="./image/age.png" width="600"/>

### 3. “找不同”注意力训练素材批量生成
自动批量生成数十组专业级“找不同”图片，支持注意力集中、视觉搜索与视空间能力训练。

<img src="./image/difference.png" width="600"/>

### 4. 多样化视觉刺激任务
同一套生成素材可灵活用于记忆力、注意力、视空间、执行功能、逻辑推理等多种认知领域训练。

<img src="./image/diversity.png" width="600"/>

### 5. 情感修复级内容：生成“从未拍过的珍贵照片”
为患者合成年轻时的结婚照、与已故亲人的合影等“缺失的记忆照片”，提供极强的情感支持与回忆触发。

<img src="./image/wedding.png" alt="年轻时的结婚照（AI生成）"  width="700" />

## Part 2：VR多感官认知与平衡康复训练

专为认知障碍兼平衡障碍患者设计，融合视觉空间、前庭觉、本体觉、感统处理的沉浸式VR训练场景。

<img src="./VR.gif?raw=true" alt="VR认知康复训练演示" />

## Part 3：个性化音视频回忆故事生成

### 核心输入（仅需三项）
- 患者及亲友的2张照片  
- 1段亲友的真实语音（用于声音克隆）  
- 简短的故事文字描述  

### 系统自动输出
完整、高情感共鸣的音视频回忆故事（含亲友声音旁白 + 患者本人形象）
